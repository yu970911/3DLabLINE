# スプレッドシート初期テンプレ（ヘッダーのみ）v1.0

Googleスプレッドシートに貼り付けるための列ヘッダーのみのテンプレ。
各シートごとにCSV形式で記載。

---

## 1) 友だち一覧

```text
LINEユーザーID,表示名,フルネーム,区分,状態,最終参加ワークショップID,最終アクション日時,初回参加日,参加回数,興味タグ,メモ
```

## 2) ワークショップ開催管理

```text
ワークショップID,ワークショップ名,開催日,開始時刻,終了時刻,当日使用URL,紹介サイトURL,リマインド送信日時,お礼送信日時,クーポン文面,配信対象タグ,有効
```

## 3) 参加者一覧（参加履歴）

```text
ワークショップID,LINEユーザーID,フルネーム,参加区分,参加日時,リマインド送信日時,お礼送信日時,アンケート回答済,備考
```

## 4) メッセージ管理

```text
メッセージID,用途,タイトル,本文,クイック返信,有効,最終更新日時,備考
```

## 5) 送信履歴

```text
送信日時,種別,宛先LINEユーザーID,宛先区分,関連ワークショップID,使用メッセージID,送信本文,送信結果,LINEレスポンス,担当
```
