# メッセージ文面 初期セット（v1.0）

メッセージ管理シートへ登録する初期文面案。
Phase1に必要な最小セット。

---

## 登録用サンプル（CSV）

```text
MSG-001,用途選択,まずは用途を選んでください,ご用件を選択してください。,当日参加:当日参加|事前登録:事前登録|情報のみ:情報のみ,TRUE,2025-02-01 09:00:00,
MSG-002,氏名入力依頼,フルネーム入力,フルネーム（例: 山田 太郎）を入力してください。, ,TRUE,2025-02-01 09:00:00,
MSG-003,当日案内,当日のご案内,こんにちは{FULL_NAME}さん。当日はこちらからご参加ください。{JOIN_URL},アンケート:アンケート|問い合わせ:問い合わせ|案内:案内,TRUE,2025-02-01 09:00:00,
MSG-004,見込み登録完了,登録完了,登録ありがとうございます。最新の案内が必要な場合は「案内」と送ってください。,案内:案内|アンケート:アンケート|問い合わせ:問い合わせ,TRUE,2025-02-01 09:00:00,
MSG-005,WS未取得エラー,開催情報未取得,現在開催情報が見つかりません。スタッフへお問い合わせください。,問い合わせ:問い合わせ|案内:案内,TRUE,2025-02-01 09:00:00,
MSG-006,用途再確認,用途を選択してください,次のボタンから選択してください。,当日参加:当日参加|事前登録:事前登録|情報のみ:情報のみ,TRUE,2025-02-01 09:00:00,
```

---

## 備考
- `MSG-002` の Quick Reply は空欄（氏名入力を促すため）
- `MSG-003` は当日参加時の返信に使用
- `MSG-004` は見込み登録完了時
- `MSG-005` はWSが見つからない場合のエラー
- `MSG-006` は用途再提示
